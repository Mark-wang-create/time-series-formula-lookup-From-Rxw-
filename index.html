时间序列公式查询

移动平均

移动平均是一种常用的时间序列平滑方法，用于分析数据中的趋势。它通过计算数据点的固定大小窗口内的平均值来消除短期波动。

公式：
    MA(t) = (x[t] + x[t-1] + ... + x[t-n+1]) / n

    其中：
    - MA(t) 是时刻 t 的移动平均值
    - x[t] 是时刻 t 的数据点
    - n 是窗口大小

指数平滑

指数平滑是一种权重递减的时间序列平滑方法。与移动平均相比，指数平滑对最近的数据点赋予更大的权重，从而更好地捕捉数据的短期变化。

公式：
    EMA(t) = α * x[t] + (1 - α) * EMA(t-1)

    其中：
    - EMA(t) 是时刻 t 的指数平滑值
    - x[t] 是时刻 t 的数据点
    - α 是平滑因子（0 < α <= 1）

自回归模型

自回归模型（AR）是一种基于时间序列自身滞后值的线性模型。它假设当前值与过去若干时刻的值之间存在线性关系。自回归模型常用于预测时间序列数据。

公式：
x[t] = c + φ₁ * x[t-1] + φ₂ * x[t-2] + ... + φ_p * x[t-p] + ε[t]
其中：
- x[t] 是时刻 t 的数据点
- c 是常数项
- φᵢ 是滞后 i 时刻的自回归系数
- p 是模型阶数
- ε[t] 是时刻 t 的误差项

自协方差（Autocovariance）：$$ \gamma_k = E[(y_t - \mu)(y_{t-k} - \mu)] $$

自相关函数（ACF）：$$ \rho_k = \frac{\gamma_k}{\gamma_0} $$

偏自相关函数（PACF）：$$ \phi_{kk} = \text{最小二乘估计}\ y_t \text{与}\ y_{t-k} \text{的回归系数，控制}\ y_{t-1}, \ldots, y_{t-k+1} $$

单位根检验（如Augmented Dickey-Fuller检验）：$$ \Delta y_t = \alpha + \beta t + \gamma y_{t-1} + \delta_1 \Delta y_{t-1} + \cdots + \delta_{p-1} \Delta y_{t-p+1} + \epsilon_t $$

简单移动平均（Simple Moving Average）：$$ SMA_t = \frac{1}{n} \sum_{i=0}^{n-1} y_{t-i} $$

指数加权移动平均（Exponential Weighted Moving Average）：$$ EWMA_t = \alpha y_t + (1 - \alpha)EWMA_{t-1},\ 0 \leq \alpha \leq 1 $$

随机游走（Random Walk）：$$ y_t = y_{t-1} + \epsilon_t $$

季节性分解（Seasonal Decomposition）：$$ y_t = T_t + S_t + R_t $$

自回归模型（Autoregressive Model, AR(p)）：$$ y_t = \alpha + \beta_1 y_{t-1} + \beta_2 y_{t-2} + \cdots + \beta_p y_{t-p} + \epsilon_t $$

移动平均模型（Moving Average Model, MA(q)）：$$ y_t = \alpha + \epsilon_t + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \cdots + \theta_q \epsilon_{t-q} $$
