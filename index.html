时间序列公式查询

移动平均

移动平均是一种常用的时间序列平滑方法，用于分析数据中的趋势。它通过计算数据点的固定大小窗口内的平均值来消除短期波动。

公式：
    MA(t) = (x[t] + x[t-1] + ... + x[t-n+1]) / n

    其中：
    - MA(t) 是时刻 t 的移动平均值
    - x[t] 是时刻 t 的数据点
    - n 是窗口大小

指数平滑

指数平滑是一种权重递减的时间序列平滑方法。与移动平均相比，指数平滑对最近的数据点赋予更大的权重，从而更好地捕捉数据的短期变化。

公式：
    EMA(t) = α * x[t] + (1 - α) * EMA(t-1)

    其中：
    - EMA(t) 是时刻 t 的指数平滑值
    - x[t] 是时刻 t 的数据点
    - α 是平滑因子（0 < α <= 1）

自回归模型

自回归模型（AR）是一种基于时间序列自身滞后值的线性模型。它假设当前值与过去若干时刻的值之间存在线性关系。自回归模型常用于预测时间序列数据。

公式：
x[t] = c + φ₁ * x[t-1] + φ₂ * x[t-2] + ... + φ_p * x[t-p] + ε[t]
其中：
- x[t] 是时刻 t 的数据点
- c 是常数项
- φᵢ 是滞后 i 时刻的自回归系数
- p 是模型阶数
- ε[t] 是时刻 t 的误差项
